### shell 与 作用域

在shell中 "{""}"为代码块
如
```shell

{
    ...
}

fun(){
    ...
}

```
其中两种形式是本质上是一样的，第一个为匿名函数，第二个为定义的函数

"("")" 特殊与代码块，它会将里面的内容新生成一个shell进行执行

也就是说如

```shell

(
test=1
)
echo "$test"

test=2
echo "$test"

{
test=3
}
echo "$test"

```

这段代码的执行结果将是

```
空
2
3
```
在shell中变量定义是一种命令，和touch、mkdir等是一样的

但是```test=1```不同，它会在一个新的shell中执行，所以对于其它部分它与其相当于是两个不同的shell

而```test=3```不会所以他会直接覆盖前一个test，也就相当于对同一个文件用">"进行覆盖
